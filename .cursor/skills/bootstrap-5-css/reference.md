# Bootstrap 5.3 documentation map

Base URL: `https://getbootstrap.com/docs/5.3/`

## GitHub doc source (MDX, AI-friendly)

Bootstrap publishes the **same** documentation as MDX under the release tag that matches the npm `bootstrap` version (this repo: **v5.3.8**).

- **Browse tree**: [twbs/bootstrap → `site/src/content/docs` @ v5.3.8](https://github.com/twbs/bootstrap/tree/v5.3.8/site/src/content/docs)
- **Raw file pattern** (replace `<section>` / `<page>` with the path inside that folder, usually `*.mdx`):

  `https://raw.githubusercontent.com/twbs/bootstrap/v5.3.8/site/src/content/docs/<section>/<page>.mdx`

Example (Vite guide source): `.../getting-started/vite.mdx`

Folder names align with the public site: `getting-started`, `layout`, `content`, `forms`, `components`, `helpers`, `utilities`, `customize`, `extend`, `about`, plus root files like `migration.mdx`.
